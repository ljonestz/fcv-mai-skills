FCV mAI Skills — Toolkit Map

World Bank FCV Group · mAI Platform (ai.worldbank.org/mai) · April 2026 (Updated)

Recent changes (April 2026): New skills added: Country Risk Scan, Country Portfolio Scanner, Portfolio Risk Impact Assessment (interlinked trio for country-level FCV and portfolio analysis), and Prompt Refiner (meta-skill for clarifying user requests). Tier structure updated: Tier 1 now 7 skills (added Country Risk Scan), Tier 2 now 5 skills (added Country Portfolio Scanner and Portfolio Risk Impact Assessment), Utility tier expanded to 3 skills (added Prompt Refiner). Total package now 15 skills. Previous changes (early April): Project Screener restructured, RRA Mini Update check-in added, AI disclaimer moved to final outputs only. ISR Flag Reviewer retired → replaced by ISR Portfolio Scanner (late March).
Tier 1 — Core Skills
Deep analytical workflows — structured, multi-step
fcv-project-screener
Project Screener
3-stage FCV screening of PADs, PCNs, PIDs, AFs and ISRs. Stage 2 produces dynamic analytical themes (derived from 12 OST recs and Do No Harm), synthesis paragraphs, and S/R ratings. Stage 3 produces structured action cards. Quick Flag mode available.
Review Analyse
fcv-cpf-peer-reviewer
CPF Peer Reviewer
4-phase peer review of CPF packages. Evidence log, gap inventory, severity classification, and 1,000–1,200 word peer review memo.
Review
fcv-rra-section-drafter
RRA Section Drafter
Collaborative drafting of RRA sections. Retrieves RRA methodology and country evidence. Iterative refinement workflow — coordinator drives, skill assists.
Draft
fcv-risk-briefing-generator
Risk Briefing Generator
Maps recent FCV developments to CPF objectives. Analyses realized and potential portfolio risks per objective. Upload CPF or provide country name.
Portfolio Analyse
fcv-country-risk-scan
Country Risk Scan
Generate a structured 500-word FCV briefing for a country covering risk overview, institutional fragility, conflict and violence dynamics, displacement, and government-development actor relations. Takes country, timeframe, and optional sector as inputs.
Analyse Draft
fcv-lens-note-drafter
FCV Lens Note Drafter
Frames FCV risks, opportunities, and design considerations for a specific sector or project engagement in a given country.
Draft Analyse
fcv-rra-mini-update
RRA Mini Update
Concise analytical update bridging an outdated RRA to the current situation. Extracts baseline, pauses for confirmation, searches for material changes, checks in on themes and format before drafting, then outputs an update note in RRA style.
Analyse Draft
Tier 2 — Productivity Skills
Flexible, faster tools — adapt to task at hand
fcv-isr-portfolio-screener
ISR Portfolio Scanner NEW
Scans multiple ISRs across a country portfolio, single project, or sector. Identifies rating inconsistencies, stale language, coverage gaps, aggregate signals, and project-derailing FCV flags. Produces a high-level situational brief.
Portfolio Analyse
fcv-lessons-learned-extractor
Lessons Learned Extractor
Multi-pass search of WBG ICRs, ISRs, and evaluations for FCV lessons. Produces a thematic synthesis note with operational implications.
Evidence Analyse
fcv-trusted-source-analyst
Trusted Source Analyst
Four modes: Verify, Enhance, Compare, Brief. Applies ACLED, ICG, OCHA, UNHCR, WBG internal and other trusted sources to any FCV task.
Evidence Review
fcv-country-portfolio-scanner
Country Portfolio Scanner
Scans a country's active World Bank portfolio by sector with project names linked to documents (PAD/ISR), funding, status, and FCV relevance flags. Outputs as sector summary table plus narrative or narrative by sector.
Portfolio Analyse
fcv-portfolio-risk-impact-assessment
Portfolio Risk Impact Assessment
Synthesizes country FCV risks with portfolio to assess how specific risks threaten specific projects. Activates Country Risk Scan + Country Portfolio Scanner if needed, then produces vulnerability matrix and adaptive management brief. Vulnerability types: field access, counterpart capacity, demand shock, supply chain, fiduciary risk, staff safety.
Portfolio Analyse
Utility Skills
Navigation and compliance — lightweight helpers
fcv-skills-overview
Skills Overview
Navigation hub. Presents all main skills in a table. User picks a number to launch a skill, or describes their task and the skill suggests the best match.
Navigation
fcv-ai-disclaimer
AI Disclaimer
Appends the standard WBG FCV AI-generated output disclaimer. Also embedded directly in every other skill — standalone use optional.
Compliance
fcv-prompt-refiner
Prompt Refiner
Meta-skill: Clarifies and refines a user prompt through up to 6 targeted follow-up questions, then generates a polished, reusable refined prompt. Use before executing complex analytical requests or when initial request is vague.
Navigation

Logical links between skills

→
Skills Overview → all main skills
Entry point: user describes task, skill routes to the right tool
→
Prompt Refiner → any analytical skill
Optional: clarify and refine a complex or vague request before activating a main skill
→
ISR Portfolio Scanner → Risk Briefing Generator
Portfolio scan surfaces FCV patterns that feed directly into a CPF-linked risk briefing
→
Country Risk Scan ↔ Country Portfolio Scanner → Portfolio Risk Impact Assessment
Risk scan and portfolio scan outputs combine in Portfolio Risk Impact Assessment to identify which projects are vulnerable to which risks
→
RRA Mini Update → RRA Section Drafter
Mini Update can hand off to Section Drafter to turn an update note into a full drafted section
→
Trusted Source Analyst → any analytical skill
Evidence retrieval (Verify / Enhance / Compare) can feed CPF Reviewer, Project Screener, Lens Note, or RRA Drafter
→
Lessons Learned Extractor → Lens Note Drafter
Lessons output can be pasted into Lens Note to ground sector framing in WBG portfolio evidence
↔
CPF Peer Reviewer ↔ Risk Briefing Generator
CPF review informs portfolio risk framing; Risk Briefing informs CPF peer review gap analysis
↔
Project Screener ↔ ISR Portfolio Scanner
Screener works at design stage on individual projects; Portfolio Scanner works at implementation stage across the portfolio
→
ISR Portfolio Scanner → management brief
Built-in Step 8 produces a CMU/management-ready brief directly from the portfolio scan output
